VisionCraze
AI-powered assistive glasses that narrate your surroundings in real time for visually impaired users.
What it does
Point a camera at your environment and VisionCraze speaks what it sees obstacles, stairs, currency, people, signs, traffic lights, terrain changes, and hazards. Built for the 285 million people worldwide living with visual impairment.
Live Demo
Open the site, click Start Camera, and point it around. It narrates everything out loud.
Tech
HTML, CSS, JavaScript, Web Speech API, MediaDevices API, AI Vision API
Setup

Open index.html
Replace YOUR_API_KEY_HERE with your API key
Deploy to Netlify
Allow camera permissions and start
